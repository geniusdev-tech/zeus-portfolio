# zeus.dev

Portfolio de Zeus — IT Professional.

- **Frontend:** React + Vite → Vercel
- **Backend:** Go API → Railway
- **AI Chat API:** Vercel Functions (`api/chat`, `api/health`) → Gemini API + MongoDB Atlas

## Scripts

- `npm run dev` inicia o frontend
- `npm run build` gera o build do frontend
- `npm run deploy:firebase` faz build + deploy no Firebase Hosting

## Vercel Chat Env Vars

- `MONGODB_URI`
- `GEMINI_API_KEY`
- `GEMINI_MODEL`
